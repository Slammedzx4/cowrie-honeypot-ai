# Cowrie Honeypot AI

SSH Honeypot with simulated attacks and AI-based detection logic. Built in a virtual lab using Cowrie, Python, and custom dashboards.

## Features
- Cowrie deployment on Ubuntu Server
- Simulated brute-force attacks from attacker VM
- JSON log parsing, EDA, and clustering detection
- Lightweight analytics dashboard

## Project Structure
- `setup/`: Installation & network configuration scripts
- `attacks/`: Simulated attack scripts
- `logs/`: Honeypot JSON logs (samples only)
- `analysis/`: Jupyter notebooks for parsing and detection
- `dashboard/`: Optional React dashboard (screenshots included)
- `paper/`: IEEE-formatted project paper

## Requirements
- Python 3.11
- pandas, numpy, matplotlib, scikit-learn
